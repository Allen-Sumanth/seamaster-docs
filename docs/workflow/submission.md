# Submission Guidelines

## File Format
Submit a single **Zip file** containing your code.

*   Must include `submission.py` (or `main.py` depending on platform rules).
*   Must define `spawn_policy`.

## Constraints
*   **Max File Size**: 5MB.
*   **No Network Access**: The sandbox blocks internet access.
*   **Libraries**: Only standard python libraries + `oceanmaster` are available. `numpy` is NOT available (unless specified otherwise).

## Verification
Before submitting, run your bot locally against the provided `simple_bot` to ensure it doesn't crash.
