# Glossary

*   **Tick**: A single turn in the game. Max 1000 ticks.
*   **Bot**: A programmable unit.
*   **Context (`ctx`)**: The data object containing current game state visible to a bot.
*   **Scraps**: Currency used to spawn bots.
*   **Energy**: Fuel for movement and abilities.
*   **Bank**: Location to deposit algae.
*   **Sensing**: The act of checking the board within Vision Radius.
*   **Manhattan Distance**: The distance metric used ( |x1-x2| + |y1-y2| ).
